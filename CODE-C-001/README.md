# CODE-C-001 :: Hello World

## Objective

Write, compile and execute the first C program.

## Files

- `hola.c` : Source code.
- `hola.o` : Object file generated by the compiler.
- `hola` : Executable generated by the linker.

## Build

Compile:

```bash
gcc -c hola.c -o hola.o
```

Link:

```bash
gcc hola.o -o hola
```

Run:

```bash
./hola
```

## Learning Goals

- Create the first C program.
- Understand the difference between source code, object files and executables.
- Compile and link a program manually.
