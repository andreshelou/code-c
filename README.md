# CODE-C

This repository contains all the examples developed throughout the **CODE-C** C programming series.

The goal of this series is not only to learn the C language, but also to understand what happens behind the scenes when a program is compiled and executed.

Topics are presented incrementally, with each chapter building on the previous one.

---

## Repository Structure

```
code-c/
├── CODE-C-001/
├── CODE-C-002/
├── CODE-C-003/
└── ...
```

Each directory corresponds to one episode of the series and contains the source code and supporting material for that chapter.

---

## Learning Philosophy

The series focuses on understanding the complete development process.

Rather than relying on IDEs or build systems from the beginning, every step is performed manually to make the compilation process explicit.

Topics include:

- The C language
- GCC
- Object files
- Linking
- Executables
- Memory
- Functions
- Pointers
- Data structures

This repository is intended to be followed together with the companion **Assembly** series.
